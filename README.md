# human_action_recognition
Code accompanying the paper "High speed human action recognition using a photonic reservoir computer" [https://doi.org/10.1016/j.neunet.2023.06.014], written by E. Picco, P. Antonik, S. Massar, and published in Neural Networks. 
Experimental code in Python (software interface) and VHDL (to program the Virtex-7 FPGA chip). 
Simulation scripts are provided in MATLAB.
